If...else statement

An if...else statment is a logical condition that checks to see if a person follows a certain rule for if to be exercuted.

Example: imagine if you want to host a contest for people of a certain age(18+) you will need to check if the person registering is of the required age. If the person is 18+, he is free to enter but if not then the person needs to go home. this example can be written in python code as follows;

age=int('enter age: ') if age >=18: print("Enter the contest") else: print("sorry you are under the required age.")

Lets say you want to add another section for kids of a particular age range in your contest, you'll, this is when we use the elif condition. Lets look at how this is done

age=int('enter age: ') if age >=18: print("Enter the contest") elif age>=17 and age<18: print("go to the kids section") else: print("sorry you are under the required age.")

A loop is like a person that keeps working around in circles.A while loop is a loop that keeps repeating itself untill a it a particular condition is false.

if you look clearly, the above code prints from 10 down to 1. The loop will stop working only when the condition x!=0 is false.Loops can also be infinite(meaning it does not end). Below is an example;

this code will print the value inside x forever.pyhton loops works with 2 statements which are the Break and continue

For loop

A for loop is used for iterating over a sequence (that is either a list, a tuple, a dictionary, a set, or a string). This is less like the for keyword in other programming languages, and works more like an iterator method as found in other object-orientated programming languages. With the for loop we can execute a set of statements, once for each item in a list, tuple, set etc.

fruits = ["apple", "banana", "cherry"] for x in fruits: print(x)

This code prints each fruit individually.
